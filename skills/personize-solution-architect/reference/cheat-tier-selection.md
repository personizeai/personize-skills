# Tier Selection Guide

| Data Type | Volume | Recommended Tier | Cost |
|-----------|--------|-----------------|------|
| Structured data (JSON, CSV fields) | Any | basic | 1 credit |
| Short text (1-2 sentences) | Any | basic | 1 credit |
| Rich text (notes, emails, 1 page) | Any | pro (default) | 1 credit |
| Long text (transcripts, reports) | Any | pro | 1 credit |
| Complex multi-topic content | Any | ultra | 1 credit |
| Batch import (CRM sync) | <100 | pro | 1/item |
| Batch import (large) | 100-10K | pro, chunked | 1/item |
| Batch import (massive) | 10K+ | Pipeline (Trigger.dev) | 1/item |

All tiers cost 1 credit per item. Tier affects extraction QUALITY, not price.
pro is the safe default. Use ultra only for content mixing multiple topics.
